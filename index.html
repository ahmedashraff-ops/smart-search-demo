📖 Setup Guide
🔑 API Key
console.yandex.cloud → Your folder → Service Accounts → Select account → API Keys → Create API Key → Copy the secret value.
📁 Folder ID
console.yandex.cloud → Click your folder name at the top → The Folder ID is on the overview page (starts with b1g...).
🤖 Agent ID (optional)
aistudio.yandex.cloud → Agent Atelier → Open your agent → Copy the ID from the URL or settings panel. Leave blank to use YandexGPT Pro directly.
🌐 Proxy URL — deploy once in 5 minutes
This is needed because browsers block direct Yandex API calls (CORS). The proxy you deploy on Vercel handles it.
Steps:
  1. Go to github.com → New repository → name it smart-search-proxy → Create
  2. Upload the api/proxy.js and vercel.json files (from the ZIP you downloaded) to that repo
  3. Go to vercel.com → Sign up free → New Project → Import your GitHub repo
  4. Click Deploy — no settings to change
  5. Vercel gives you a URL like smart-search-proxy.vercel.app
  6. Paste https://smart-search-proxy.vercel.app/api/proxy into the Proxy URL field above
